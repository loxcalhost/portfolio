```
def encryption(msg):
    ct = []
    for char in msg:
        ct.append((123 * char + 18) % 256)
    return bytes(ct)
```

The function `encryption(msg)` takes `msg` as input and initializes an empty list ct to store encrypted values.

A `for` loop iterates over each byte (`char`) in `msg`. For each value, the encryption formula `(123 * char + 18) % 256` is applied. This means the byte is first multiplied by 123, then 18 is added, and finally the result is reduced modulo 256 to keep it within the byte range (0–255).

Each encrypted value is appended to the list `ct`. Finally, the list is converted into a bytes object using `bytes(ct)` and returned.


The function `encryption(MSG)` is called to encrypt the original message, and the result is stored in the variable `ct`.

A file named `msg.enc` is then opened in write mode (`'w'`). The encrypted bytes stored in ct are converted into a hexadecimal string using `.hex()` and written into the file.

Finally, the file is closed using `f.close()` to ensure the data is properly saved.

```
with open("ct.bin", "rb") as file:
    ct = file.read()

inv = pow(123, -1, 256)

pt = []
for y in ct:
    x = ((y - 18) * inv) % 256
    pt.append(x)

pt = bytes(pt)

print(pt)
```

**Code Breakdown**

```
with open("ct.bin", "rb") as file:
    ct = file.read()
```

`with open("ct.bin", "rb") as file:` opens the `ct.bin` file in **binary read mode** **`rb`** and second line reads entire file content and store it in `ct`, now `ct` contains encrypted data as a `bytes object`, which allows us to process each byte during decryption.

> **NOTE:** We need binary mode (`b`) here because our file contains **encrypted bytes**, not text

```
inv = pow(123, -1, 256)
```

In Python, `pow(a, b, m)` usually means `$a^b$ mod m`, so when `b= -1` Python computes the modular inverse of `a` mod `m`. So, `pow(123, -1, 256)` means find number `inv` such that `(123 × inv) % 256 = 1`

Therefore, our decryption becomes, `x = ((y - 18) * inv) % 256`

```
pt = []
for y in ct:
    x = ((y - 18) * inv) % 256
    pt.append(x)
```

`pt = []` is used to create an empty list that will store **decrypted byte values**

A `for` loop iterates over each byte `y` in `ct`, and the decryption operation is applied to recover the original byte.

`pt.append(x)` this adds the decrypted byte `x` to the list

```
pt = bytes(pt)

print(pt)
```

`pt = bytes(pt)` here `bytes(pt)` converts the list into **byte objects**

`print(pt)` prints the byte string
